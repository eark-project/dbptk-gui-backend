1. Connect to database

Select your database management system (DBMS) and configure the connection.

DBMS is your database management system. If you can't find yours here, contact us.
Hostname is the name or IP address of the machine where your DBMS is runing.
Port is communication endpoint of your DBMS. If you don't know, try the default one.
User name is the name of the user that has persmission to access the database.
User password is the password of the user that has permission to access the database.
